11. Objetos e Classes

Python é uma linguagem orientada a objetos, o que significa que você pode criar classes e objetos para organizar e estruturar o código.

Criando uma Classe

Uma classe é uma espécie de molde para criar objetos. Ela define atributos e comportamentos para os objetos criados a partir dela.

Exemplo:

class Carro:
	def __init__(self, modelo, cor):
		self.modelo = modelo  # Atributo
		self.cor = cor        # Atributo
        		
	def dirigir(self):
		print(f"Dirigindo o {self.modelo} de cor {self.cor}.")
        				
# Criando um objeto da classe Carro
meu_carro = Carro("Fusca", "azul")
meu_carro.dirigir()  # Saída: Dirigindo o Fusca de cor azul.

Construtores e o método __init__

O método __init__ é o construtor da classe, que é chamado sempre que você cria um objeto.

Herança

Python permite que você crie classes que herdam comportamentos e atributos de outras classes.

Exemplo de herança:

class Veiculo:
	def __init__(self, modelo):
		self.modelo = modelo
        			
	def andar(self):
		print(f"O {self.modelo} está andando.")
        				
class Carro(Veiculo):
	def __init__(self, modelo, cor):
		super().__init__(modelo)  # Chama o construtor da classe mãe
		self.cor = cor
        		
	def buzinar(self):
		print("Buzinando!")
        		
# Criando um objeto da classe Carro
carro = Carro("Fusca", "azul")
carro.andar()  # Saída: O Fusca está andando.
carro.buzinar()  # Saída: Buzinando!

Encapsulamento e Atributos Privados

O encapsulamento permite esconder certos atributos de uma classe para que eles não sejam acessados diretamente.

Exemplo:

class ContaBancaria:
	def __init__(self, saldo_inicial):
		self.__saldo = saldo_inicial  # Atributo privado
        			
	def depositar(self, valor):
		self.__saldo += valor
        			
	def consultar_saldo(self):
		print(f"Saldo atual: R${self.__saldo}")
        		
conta = ContaBancaria(1000)
conta.depositar(500)
conta.consultar_saldo()  # Saída: Saldo atual: R$1500

12. Bibliotecas para Análise de Dados

Python é amplamente utilizado em ciência de dados e análise de dados devido às suas bibliotecas poderosas, como Pandas, NumPy, Matplotlib e Seaborn.

Pandas

O Pandas é uma biblioteca usada para manipulação e análise de dados, oferecendo estruturas de dados rápidas e flexíveis.

Exemplo básico de uso do Pandas:

import pandas as pd
        		
# Criando um DataFrame (estrutura de dados bidimensional)
dados = {"Nome": ["Alice", "Bob", "Charlie"], "Idade": [25, 30, 35]}
df = pd.DataFrame(dados)
        		
print(df)

Saída:

      Nome  Idade
0    Alice     25
1      Bob     30
2  Charlie     35

Manipulação de Dados com Pandas

Você pode realizar operações como filtrar, agrupar, ordenar e calcular estatísticas de dados de forma simples:

# Filtrando dados
df_filtrado = df[df["Idade"] > 25]
print(df_filtrado)
        		
# Calculando a média de idade
media_idade = df["Idade"].mean()
print(f"Média de idade: {media_idade}")

NumPy

O NumPy é uma biblioteca usada para computação numérica em Python, fornecendo suporte para arrays e operações matemáticas avançadas.

Exemplo básico:

import numpy as np
        		
# Criando um array NumPy
arr = np.array([1, 2, 3, 4, 5])
print(arr + 10)  # Saída: [11 12 13 14 15]

Matplotlib e Seaborn

O Matplotlib e o Seaborn são bibliotecas populares para visualização de dados.

Exemplo com Matplotlib:

import matplotlib.pyplot as plt
        		
# Criando um gráfico de linha simples
x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]
plt.plot(x, y)
plt.title("Gráfico de Linha")
plt.xlabel("x")
plt.ylabel("y")
plt.show()

Exemplo com Seaborn:

import seaborn as sns
        		
# Criando um gráfico de dispersão com Seaborn
dados = sns.load_dataset("iris")
sns.scatterplot(data=dados, x="sepal_length", y="sepal_width", hue="species")
plt.show()

13. Manipulação de Strings

As strings são um dos tipos de dados mais comuns em Python. Elas podem ser manipuladas de diversas formas usando métodos embutidos ou operadores.

Métodos Comuns para Manipulação de Strings

  • lower() e upper(): Convertem a string para minúsculas ou maiúsculas.
texto = "Python é Legal"
print(texto.lower())  # Saída: python é legal
print(texto.upper())  # Saída: PYTHON É LEGAL
  • strip(): Remove espaços em branco no início e no final da string.
texto = "  Olá Mundo!  "
print(texto.strip())  # Saída: Olá Mundo!
  • replace(): Substitui uma substring por outra.
texto = "Eu amo Python"
texto_substituido = texto.replace("Python", "programar")
print(texto_substituido)  # Saída: Eu amo programar
  • split(): Divide uma string em uma lista de substrings.
texto = "Python, Java, C++"
linguagens = texto.split(", ")
print(linguagens)  # Saída: ['Python', 'Java', 'C++']
  • join(): Junta uma lista de strings em uma única string.
linguagens = ['Python', 'Java', 'C++']
texto = ", ".join(linguagens)
print(texto)  # Saída: Python, Java, C++

F-strings (Interpolação de Strings)

As f-strings permitem inserir valores dentro de strings de maneira mais legível e eficiente:

nome = "Alice"
idade = 30
mensagem = f"Meu nome é {nome} e tenho {idade} anos."
print(mensagem)  # Saída: Meu nome é Alice e tenho 30 anos.

Busca em Strings

Você pode buscar por substrings dentro de strings usando o método find() e outras alternativas:

texto = "Python é uma linguagem poderosa"
posicao = texto.find("linguagem")
print(posicao)  # Saída: 10 (posição da substring)

14. Funções Avançadas

Funções em Python não são limitadas aos exemplos básicos. Aqui estão alguns recursos mais avançados que podem ser muito úteis.

Funções Anônimas (Lambda)

As funções lambda são funções pequenas e anônimas, criadas com a palavra-chave lambda: 

soma = lambda x, y: x + y
print(soma(3, 5))  # Saída: 8

Elas são frequentemente usadas com funções como map(), filter(), e reduce().

Função map()

A função map() aplica uma função a todos os itens de uma lista ou qualquer iterável: 

numeros = [1, 2, 3, 4, 5]
quadrados = list(map(lambda x: x**2, numeros))
print(quadrados)  # Saída: [1, 4, 9, 16, 25]

Função filter()

A função filter() filtra os elementos de uma sequência com base em uma função:

numeros = [1, 2, 3, 4, 5, 6]
pares = list(filter(lambda x: x % 2 == 0, numeros))
print(pares)  # Saída: [2, 4, 6]

Função reduce()

A função reduce() aplica uma função de forma acumulativa a todos os elementos de um iterável. Ela está disponível no módulo functools:

from functools import reduce
numeros = [1, 2, 3, 4]
produto = reduce(lambda x, y: x * y, numeros)
print(produto)  # Saída: 24 (1 * 2 * 3 * 4)

Funções de Ordem Superior

Funções de ordem superior são aquelas que podem receber outras funções como argumentos ou retornar funções como resultado. Por exemplo:

def operacao(func, a, b):
	return func(a, b)
        		
soma = lambda x, y: x + y
produto = lambda x, y: x * y
        		
print(operacao(soma, 3, 5))  # Saída: 8
print(operacao(produto, 3, 5))  # Saída: 15

15. Expressões Regulares

As expressões regulares (regex) são usadas para encontrar padrões de texto em uma string. Python oferece a biblioteca re para trabalhar com regex.

Fundamentos de Expressões Regulares

Uma expressão regular é uma sequência de caracteres que forma um padrão de busca. Por exemplo, \d representa um dígito.

  • Buscar um padrão simples:
import re
texto = "Meu número é 12345"
resultado = re.search(r"\d+", texto)  # Procura por números
if resultado:
	print(resultado.group())  # Saída: 12345

Funções Comuns de Regex

  • search(): Busca o padrão em qualquer lugar da string.
  • match(): Verifica se o padrão está no início da string.
  • findall(): Encontra todas as ocorrências do padrão em uma string.
texto = "Email: exemplo@dominio.com e contato@dominio.com"
emails = re.findall(r'\b\w+@\w+\.\w+\b', texto)  # Busca por emails
print(emails)  # Saída: ['exemplo@dominio.com', 'contato@dominio.com']

Substituição com sub()

A função sub() substitui o padrão por uma string fornecida:

texto = "Meu número de telefone é 12345"
novo_texto = re.sub(r"\d+", "#####", texto)
print(novo_texto)  # Saída: Meu número de telefone é #####

Padrões Comuns

  • \d: Um dígito.
  • \w: Qualquer caractere alfanumérico.
  • .: Qualquer caractere.
  • ^: Início da string.
  • $: Fim da string.

16. Compreensão de Listas

A compreensão de listas é uma forma concisa de criar listas, aplicando uma expressão a cada item de uma sequência.

Compreensão Básica

Você pode criar uma lista aplicando uma expressão a cada elemento de uma sequência:

quadrados = [x**2 for x in range(10)]
print(quadrados)  # Saída: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

Compreensão com Condições

Você também pode incluir condições para filtrar elementos:

pares = [x for x in range(10) if x % 2 == 0]
print(pares)  # Saída: [0, 2, 4, 6, 8]

Compreensão de Dicionários

Você pode criar dicionários de maneira semelhante às listas:

dicionario = {x: x**2 for x in range(5)}
print(dicionario)  # Saída: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

Compreensão de Conjuntos

Conjuntos também podem ser criados de maneira semelhante às listas:

quadrados = {x**2 for x in range(10)}
print(quadrados)  # Saída: {0, 1, 4, 9, 16, 25, 36, 49, 64, 81}

17. Trabalhando com Banco de Dados

Python oferece diversas bibliotecas para interagir com bancos de dados, tanto relacionais quanto não relacionais. Vamos ver um exemplo com SQLite, que é um banco de dados embutido no Python, e SQLAlchemy, que é um ORM (Object-Relational Mapping).

Usando SQLite

SQLite é um banco de dados relacional embutido, o que significa que ele armazena dados em arquivos locais.

  • Conectando ao banco de dados e criando uma tabela:
import sqlite3
        		
# Conectando ao banco de dados (será criado se não existir)
conn = sqlite3.connect("meu_banco.db")
c = conn.cursor()
        		
# Criando uma tabela
c.execute('''CREATE TABLE IF NOT EXISTS clientes (
				id INTEGER PRIMARY KEY,
				nome TEXT,
				idade INTEGER
			)''')
conn.commit()
  • Inserindo Dados:
c.execute("INSERT INTO clientes (nome, idade) VALUES (?, ?)", ("Alice", 30))
conn.commit()
  • Consultando Dados:
c.execute("SELECT * FROM clientes")
clientes = c.fetchall()
print(clientes)  # Exemplo de saída: [(1, 'Alice', 30)]
  • Fechando a Conexão:
conn.close()

Usando SQLAlchemy (ORM)

SQLAlchemy é uma biblioteca que facilita a interação com bancos de dados através de objetos Python, eliminando a necessidade de escrever SQL diretamente.

  • Definindo um Modelo:
from sqlalchemy import create_engine, Column, Integer, String
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import sessionmaker
        		
Base = declarative_base()
        		
class Cliente(Base):
	__tablename__ = 'clientes'
	id = Column(Integer, primary_key=True)
	nome = Column(String)
	idade = Column(Integer)
        		
# Criando uma instância de engine para conectar ao banco
engine = create_engine('sqlite:///meu_banco.db')
        		
# Criando as tabelas no banco
Base.metadata.create_all(engine)
        		
# Criando uma seção para manipulação de dados
Session = sessionmaker(bind=engine)
session = Session()
        		
# Adicionando um novo cliente
novo_cliente = Cliente(nome="Bob", idade=28)
session.add(novo_cliente)
session.commit()

18. Programação Assíncrona

A programação assíncrona é importante para executar tarefas concorrentes sem bloquear o fluxo de execução do programa, muito útil para aplicativos web, redes e outras tarefas que envolvem I/O (entrada/saída).

Biblioteca asyncio

A biblioteca asyncio fornece um framework para programação assíncrona no Python, permitindo que funções sejam executadas de forma não bloqueante.

  • Função Assíncrona:
import asyncio
        		
async def minha_tarefa():
	print("Iniciando tarefa")
	await asyncio.sleep(2)  # Simula uma tarefa assíncrona
	print("Tarefa concluída")
        		
# Executando a função assíncrona
asyncio.run(minha_tarefa())

Execução Concorrente com asyncio.gather()

Se você precisar executar múltiplas tarefas assíncronas de forma concorrente:

async def tarefa_1():
	print("Tarefa 1 iniciada")
	await asyncio.sleep(1)
	print("Tarefa 1 concluída")
        		
async def tarefa_2():
	print("Tarefa 2 iniciada")
	await asyncio.sleep(2)
	print("Tarefa 2 concluída")
        		
# Executando as tarefas de forma concorrente
async def main():
	await asyncio.gather(tarefa_1(), tarefa_2())
        		
asyncio.run(main())

19. Trabalhando com Redes

Python possui diversas bibliotecas para lidar com redes, como o socket e o requests, que são muito úteis para comunicação de rede e chamadas HTTP.

Biblioteca socket

A biblioteca socket permite que você crie servidores e clientes de rede.

  • Servidor TCP simples:
import socket
        		
# Criando o socket
servidor = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
servidor.bind(("localhost", 12345))
servidor.listen(1)
        		
print("Aguardando conexão...")
cliente, endereco = servidor.accept()
print(f"Conectado a {endereco}")
        		
# Enviando dados ao cliente
cliente.send(b"Olá, cliente!")
        		
# Fechando a conexão
cliente.close()
  • Cliente TCP simples:
import socket
        		
# Criando o socket
cliente = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
cliente.connect(("localhost", 12345))
        		
# Recebendo dados do servidor
dados = cliente.recv(1024)
print(dados.decode())  # Exibe: Olá, cliente!
        		
# Fechando a conexão
cliente.close()

Biblioteca requests

A biblioteca requests facilita o envio de requisições HTTP para servidores, muito usada para trabalhar com APIs RESTful.

  • Enviando uma requisição GET:
import requests
        		
response = requests.get("https://api.github.com")
print(response.status_code)  # Código de status HTTP
print(response.json())  # Resposta em formato JSON
  • Enviando uma requisição POST:
data = {'name': 'Alice', 'age': 25}
response = requests.post("https://api.exemplo.com", json=data)
print(response.text)

20. Frameworks para Web

Python é amplamente usado para criar aplicações web, e alguns dos frameworks mais populares são Flask e Django.

Flask

Flask é um microframework que é simples de aprender e usado para aplicações web menores e mais flexíveis.

  • Exemplo de uma aplicação simples com Flask:
from flask import Flask
        		
app = Flask(__name__)
        		
@app.route('/')
def home():
	return "Olá, Mundo!"
        		
if __name__ == "__main__":
	app.run(debug=True)

Django

Django é um framework web mais robusto, usado para construir aplicações mais complexas e escaláveis. Ele segue o padrão MVC (Model-View-Controller).

  • Iniciando um Projeto Django:
  1. Instalar o Django:
pip install django
  1. Criar um novo projeto:
django-admin startproject meu_site
  1. Rodar o servidor de desenvolvimento:
python manage.py runserver

Django vem com muitas funcionalidades prontas, como autenticação de usuário, ORM, e painel administrativo.
