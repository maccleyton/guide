O HTML (HyperText Markup Language) é a base de toda página da web. Ele define a estrutura e o conteúdo de um site, servindo como o primeiro passo para quem deseja ingressar no desenvolvimento web. Este guia foi criado para ajudá-lo a dominar o HTML, começando com os fundamentos e avançando até práticas modernas e técnicas avançadas.

1. Estrutura Básica de um Documento HTML

Todo documento HTML segue uma estrutura básica que inclui tags essenciais:

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Título da Página</title>
</head>
<body>
  <h1>Bem-vindo ao HTML</h1>
  <p>Este é um exemplo de estrutura básica de um documento HTML.</p>
</body>
</html>

1.1. Explicação das Tags

  • <!DOCTYPE html>: Declara que o documento é HTML5.
  • <html>: Envolve todo o conteúdo da página.
    • Atributo lang: Define o idioma do documento.
  • <head>: Contém metadados e informações sobre a página.
    • <meta charset="UTF-8">: Define a codificação de caracteres como UTF-8.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Garante que a página seja responsiva.
    • <title>: Define o título da página que aparece na aba do navegador.
  • <body>: Contém o conteúdo visível da página.

2. Elementos de Texto

O HTML oferece várias tags para definir textos e seu formato.

2.1 Títulos

Usados para destacar cabeçalhos. Existem seis níveis: <h1> a <h6>.

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Seção</h3>
<h4>Subseção</h4>
<h5>Nota</h5>
<h6>Pequena Observação</h6>

2.2 Parágrafos e Quebras de Linha

  • <p>: Define um parágrafo.
  • <br>: Insere uma quebra de linha.
<p>Este é um parágrafo.</p>
<p>Outro parágrafo com<br>uma quebra de linha.</p>

2.3 Texto Formatado

Elemento Descrição Exemplo
<strong> Enfatiza (texto em negrito). <strong>Importante</strong>
<em> Enfatiza (texto em itálico). <em>Destaque</em>
<u> Texto sublinhado. <u>Exemplo</u>
<small> Texto menor. <small>Nota</small>
<mark> Destaca texto (marcado). <mark>Importante</mark>
<del> Texto tachado (removido). <del>Removido</del>
<ins> Texto inserido (sublinhado por padrão). <ins>Adicionado</ins>
<sup> Texto sobrescrito. E = mc<sup>2</sup>
<sub> Texto subscrito. H<sub>2</sub>O
<code> Texto formatado como código. <code>código</code>
<pre> Texto pré-formatado. <pre>texto</pre>

3. Listas

3.1 Listas Não Ordenadas

Usam o elemento <ul> com itens definidos por <li>.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

3.2 Listas Ordenadas

Usam o elemento <ol> com itens definidos por <li>.

<ol>
  <li>Primeiro Item</li>
  <li>Segundo Item</li>
  <li>Terceiro Item</li>
</ol>

3.3 Listas Aninhadas

Combina listas ordenadas e não ordenadas.

<ul>
  <li>Item Principal
    <ul>
      <li>Subitem 1</li>
      <li>Subitem 2</li>
    </ul>
  </li>
  <li>Outro Item</li>
</ul>

4. Links e Navegação

4.1 Links Simples

Usam a tag <a> com o atributo href.

<a href="https://www.exemplo.com">Visite nosso site</a>

4.2 Links com Alvo (target)

Valor Descrição
_self Abre o link na mesma janela (padrão).
_blank Abre o link em uma nova aba.
_parent Abre no frame pai.
_top Abre no contexto principal (ignora frames). 
<a href="https://www.exemplo.com" target="_blank">Abrir em nova aba</a>

4.3 Links para âncoras

Usam o atributo id para navegar para partes específicas da página.

<h2 id="seção1">Seção 1</h2>
<a href="#seção1">Ir para a Seção 1</a>

5. Imagens

5.1 Tag Básica de Imagem

Usa a tag <img> com os atributos src e alt.

<img src="imagem.jpg" alt="Descrição da imagem">

5.2 Atributos Comuns

Atributo Descrição
src URL da imagem.
alt Texto alternativo para acessibilidade.
width Largura da imagem (em pixels ou %).
height Altura da imagem (em pixels ou %).
title Texto exibido ao passar o mouse. 
<img src="imagem.jpg" alt="Descrição" width="300" height="200" title="Exemplo de Imagem">
