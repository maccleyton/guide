Após cobrir os fundamentos do HTML, esta seção explora tópicos mais avançados, incluindo formulários, tabelas e elementos multimídia. Esses recursos são essenciais para construir páginas interativas e informativas.

1. Formulários HTML

Os formulários permitem coletar dados do usuário. A tag principal é <form>, que envolve os elementos de entrada.

1.1 Estrutura Básica de um Formulário

<form action="/enviar-dados" method="POST">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <button type="submit">Enviar</button>
</form>

1.2 Atributos Comuns da Tag <form>

Atributo Descrição
action URL para onde os dados do formulário serão enviados.
method Método HTTP para envio (GET ou POST).
enctype Tipo de codificação (ex.: multipart/form-data para uploads).

1.3 Tipos de Campos de Entrada (<input>)

Tipo Descrição Exemplo
text Campo de texto comum. <input type="text">
email Valida emails automaticamente. <input type="email">
password Campo de senha (esconde os caracteres). <input type="password">
number Aceita apenas números. <input type="number">
date Seletor de datas. <input type="date">
checkbox Caixa de seleção. <input type="checkbox">
radio Botão de opção (seleção única por grupo). <input type="radio">
file Upload de arquivos. <input type="file">
submit Botão para envio do formulário. <input type="submit">

1.4 Outros Elementos de Formulário

  • <textarea>: Para textos longos.

    <textarea name="mensagem" rows="4" cols="50">Escreva aqui...</textarea>

  • <select>: Menu suspenso com opções.

    <label for="opcao">Escolha:</label>
<select id="opcao" name="opcao">
  <option value="1">Opção 1</option>
  <option value="2">Opção 2</option>
</select>

2. Tabelas

As tabelas exibem dados de forma tabular, usando a tag <table> e suas tags relacionadas.

2.1 Estrutura Básica de uma Tabela

<table border="1">
  <thead>
    <tr>
      <th>Nome</th>
      <th>Email</th>
      <th>Idade</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>João</td>
      <td>joao@example.com</td>
      <td>25</td>
    </tr>
    <tr>
      <td>Maria</td>
      <td>maria@example.com</td>
      <td>30</td>
    </tr>
  </tbody>
</table>

2.2 Tags Importantes para Tabelas

Tag Descrição
<table> Define a tabela.
<thead> Define o cabeçalho da tabela.
<tbody> Define o corpo da tabela.
<tr> Define uma linha na tabela.
<th> Define uma célula de cabeçalho (negrito por padrão).
<td> Define uma célula de dados.
<caption> Adiciona uma legenda à tabela.

2.3 Mesclagem de Células

  • colspan: Mescla células horizontalmente.
  • rowspan: Mescla células verticalmente.
<table border="1">
  <tr>
    <td colspan="2">Célula Mesclada Horizontalmente</td>
  </tr>
  <tr>
    <td rowspan="2">Célula Mesclada Verticalmente</td>
    <td>Outra Célula</td>
  </tr>
  <tr>
    <td>Célula Adicional</td>
  </tr>
</table>

3. Elementos Multimídia

O HTML suporta integração de mídia como áudio, vídeo e gráficos vetoriais (SVG).

3.1 Áudio

Usa a tag <audio> para reproduzir arquivos de áudio.

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Seu navegador não suporta o elemento de áudio.
</audio>

3.2 Vídeo

Usa a tag <video> para reproduzir arquivos de vídeo.

<video controls width="600">
  <source src="video.mp4" type="video/mp4">
  Seu navegador não suporta o elemento de vídeo.
</video>

3.3 SVG (Scalable Vector Graphics)

Permite criar gráficos vetoriais escaláveis diretamente no HTML.

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

Com esses tópicos avançados, você pode criar páginas mais interativas e funcionais.
