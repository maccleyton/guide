JavaScript é uma das linguagens de programação mais importantes do desenvolvimento web. Com ele, você pode transformar páginas estáticas em experiências interativas, dinâmicas e poderosas. Este guia foi elaborado para levar você desde os fundamentos até conceitos avançados, permitindo que você desenvolva aplicações robustas para a web e além.

1. Variáveis

Variáveis armazenam valores para uso no código. Em JavaScript, há três maneiras principais de declarar variáveis: var, let, e const.

1.1 var

  • Escopo: Função ou global.
  • Pode ser redeclarada.
  • Pode ser atualizada.
  • Problemas: Não respeita o escopo de bloco.

Exemplo:

var mensagem = "Olá";
var mensagem = "Oi de novo"; // Redeclaração permitida
console.log(mensagem); // Oi de novo

1.2 let

  • Escopo: Bloco.
  • Não pode ser redeclarada no mesmo escopo.
  • Pode ser atualizada.

Exemplo:

let nome = "Maria";
nome = "João"; // Atualização permitida
console.log(nome); // João

1.3 const

  • Escopo: Bloco.
  • Não pode ser redeclarada.
  • Não pode ser atualizada.

Exemplo:

const PI = 3.14;
// PI = 3.15; // Erro: Não é possível atualizar uma constante
console.log(PI); // 3.14

2. Tipos de Dados

Os tipos de dados em JavaScript se dividem em primitivos e objetos.

2.1 Tipos Primitivos

  • string: Sequência de caracteres.

    let saudacao = "Olá, mundo!";

  • number: Números inteiros ou de ponto flutuante.

    let idade = 30;
let preco = 19.99;

  • boolean: Verdadeiro (true) ou falso (false).

    let ativo = true;
let logado = false;

  • undefined: Variável declarada sem valor atribuído.

    let valor;
console.log(valor); // undefined

  • null: Representa ausência intencional de valor.

    let resultado = null;

  • symbol: Identificador único.

    let id = Symbol("id");

  • bigint: Números inteiros muito grandes.

    let numeroGrande = 12345678901234567890n;

2.2 Tipos de Objetos

  • Objetos Literais: Conjuntos de pares chave-valor.

    let pessoa = { nome: "João", idade: 25 };

  • Arrays: Lista ordenada de elementos.

    let frutas = ["Maçã", "Banana", "Laranja"];

  • Funções: Blocos de código executáveis.

    function somar(a, b) {
  return a + b;
}

3. Operadores

3.1 Aritméticos

Executam cálculos matemáticos.

  • + Soma
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Módulo (resto da divisão)
  • ** Exponenciação

Exemplo:

let resultado = 5 + 3; // 8

3.2 Comparação

Comparam dois valores.

  • == Igualdade (não estrita)
  • === Igualdade estrita
  • != Diferença
  • !== Diferença estrita
  • > Maior que
  • < Menor que
  • >= Maior ou igual
  • <= Menor ou igual

Exemplo:

let teste = 5 === "5"; // false

3.3 Lógicos

Operadores para lógica booleana.

  • && E
  • || Ou
  • ! Não

Exemplo:

let ativo = true && false; // false

3.4 Atribuição

Atribuem valores a variáveis.

  • = Atribuição
  • += Soma e atribuição
  • -= Subtração e atribuição
  • *= Multiplicação e atribuição
  • /= Divisão e atribuição
  • %= Módulo e atribuição

Exemplo:

let x = 10;
x += 5; // x agora é 15

3.5 Outros Operadores

  • Ternário: condicao ? valor1 : valor2

    let mensagem = idade >= 18 ? "Maior de idade" : "Menor de idade";

  • Spread: Expande elementos de arrays ou objetos.

    let numeros = [1, 2, 3];
let novosNumeros = [...numeros, 4, 5];

  • Destructuring: Extrai valores de arrays ou objetos.

    let { nome, idade } = pessoa;

4. Estruturas de Controle

4.1 Condicionais

  • if, else if, else:

    if (idade >= 18) {
  console.log("Maior de idade");
} else {
  console.log("Menor de idade");
}

  • switch:

    switch (cor) {
  case "vermelho":
    console.log("Pare");
    break;
  default:
    console.log("Continue");
}

4.2 Laços de Repetição

  • for:

    for (let i = 0; i < 5; i++) {
  console.log(i);
}

  • while:

    let x = 0;
while (x < 3) {
  console.log(x);
  x++;
}

  • do-while:

    let y = 0;
do {
  console.log(y);
  y++;
} while (y < 3);

5. Funções

Funções são blocos de código reutilizáveis.

5.1 Declaração de Funções

  • Função Nomeada:

    function saudacao(nome) {
  return `Olá, ${nome}!`;
}
console.log(saudacao("João")); // Olá, João!

  • Função Anônima:

    const somar = function (a, b) {
  return a + b;
};
console.log(somar(5, 3)); // 8

  • Arrow Function:

    const multiplicar = (a, b) => a * b;
console.log(multiplicar(4, 2)); // 8

5.2 Funções com Parâmetros Padrão

function cumprimentar(nome = "Visitante") {
  return `Olá, ${nome}!`;
}
console.log(cumprimentar()); // Olá, Visitante!

5.3 Funções Recursivas

function fatorial(n) {
  return n === 0 ? 1 : n * fatorial(n - 1);
}
console.log(fatorial(5)); // 120

6. Manipulação do DOM

O Document Object Model (DOM) permite manipular elementos HTML e seus estilos através do JavaScript.

6.1 Seleção de Elementos

  • document.getElementById: Seleciona um elemento pelo seu ID.

    let titulo = document.getElementById("titulo");

  • document.querySelector: Seleciona o primeiro elemento que corresponde ao seletor CSS.

    let paragrafo = document.querySelector("p");

  • document.querySelectorAll: Seleciona todos os elementos que correspondem ao seletor CSS.

    let itens = document.querySelectorAll("li");

6.2 Manipulação de Conteúdo

  • innerHTML: Define ou obtém o conteúdo HTML de um elemento.

    titulo.innerHTML = "Novo Título";

  • textContent: Define ou obtém apenas o texto de um elemento.
