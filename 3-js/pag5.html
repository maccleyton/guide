Aqui está um guia detalhado sobre cada um desses conceitos do JavaScript:

1. Operadores de Atribuição

Os operadores de atribuição são usados para atribuir valores a variáveis.

Operador Exemplo Equivalente a... Explicação
= x = 10 x = 10 Atribui 10 à variável x
+= x += 5 x = x + 5 Soma e atribui
-= x -= 3 x = x - 3 Subtrai e atribui
*= x *= 2 x = x * 2 Multiplica e atribui
/= x /= 2 x = x / 2 Divide e atribui
%= x %= 3 x = x % 3 Atribui o resto da divisão
**= x **= 2 x = x ** 2 Atribui potência

2. Incremento e Decremento

São usados para aumentar ou diminuir valores de variáveis numéricas.

Operador Exemplo Explicação
++ x++ Incrementa x em 1 (pós-incremento)
-- x-- Decrementa x em 1 (pós-decremento)
++x ++x Incrementa x antes de usar o valor
--x --x Decrementa x antes de usar o valor

Exemplo:

let a = 5;
console.log(a++); // Exibe 5 e depois incrementa para 6
console.log(++a); // Incrementa primeiro para 7 e depois exibe 7

3. Concatenar Strings e Variáveis

A concatenação une strings com outras strings ou variáveis.

Exemplo:

let nome = "Maria";
let mensagem = "Olá, " + nome + "! Bem-vinda.";
console.log(mensagem);

4. Interpolação de String (Template Literals)

A interpolação usa crases (``) e ${} para inserir valores dentro de uma string.

Exemplo:

let nome = "João";
let idade = 30;
console.log(`Meu nome é ${nome} e eu tenho ${idade} anos.`);

5. Operador typeof

O typeof retorna o tipo de dado de uma variável.

Exemplo:

console.log(typeof "Olá"); // string
console.log(typeof 10);    // number
console.log(typeof true);  // boolean
console.log(typeof {});    // object
console.log(typeof null);  // object (erro histórico do JS)
console.log(typeof undefined); // undefined

6. Declarações Condicionais (if, if...else, else if)

Usadas para tomar decisões no código.

Exemplo:

let idade = 18;
if (idade < 18) {
  console.log("Menor de idade");
} else if (idade === 18) {
  console.log("Tem exatamente 18 anos");
} else {
  console.log("Maior de idade");
}

7. Operadores de Comparação

Usados para comparar valores.

Operador Descrição Exemplo (a = 5, b = "5") Retorno
== Igualdade de valor (não compara tipo) a == b true
=== Igualdade estrita (compara tipo) a === b false
!= Diferente de valor a != b false
!== Diferente de valor e tipo a !== b true
> Maior que a > 3 true
< Menor que a < 3 false
>= Maior ou igual a >= 5 true
<= Menor ou igual a <= 5 true

8. Operadores Lógicos

Usados para combinar expressões.

Operador Descrição Exemplo
&& E (AND) (a > 0 && b < 10) (ambos devem ser true)
` `
! Negação (NOT) !(a > 0) (inverte o valor lógico)

9. Operador Ternário

Uma forma simplificada de if...else.

Exemplo:

let idade = 20;
let status = (idade >= 18) ? "Maior de idade" : "Menor de idade";
console.log(status);

10. switch

Substitui if...else quando há múltiplas opções.

Exemplo:

let cor = "azul";
switch (cor) {
  case "vermelho":
    console.log("A cor é vermelha");
    break;
  case "azul":
    console.log("A cor é azul");
    break;
  default:
    console.log("Cor não reconhecida");
}

11. Funções no JavaScript

Funções são blocos de código reutilizáveis.

🔹 Exemplo:

function saudacao(nome) {
  return `Olá, ${nome}!`;
}
console.log(saudacao("Carlos"));

12. Parâmetros e Argumentos

  • Parâmetros: valores que uma função recebe.
  • Argumentos: valores passados na chamada da função.

13. Parâmetros Padrão

Caso um argumento não seja passado, um valor padrão pode ser usado.

Exemplo:

function cumprimentar(nome = "Visitante") {
  console.log(`Olá, ${nome}!`);
}
cumprimentar(); // "Olá, Visitante!"

14. return (Retorno de Função)

Faz a função retornar um valor.

Exemplo:

function dobro(num) {
  return num * 2;
}
console.log(dobro(4)); // 8

15. Expressões de Função

Uma função anônima atribuída a uma variável.

Exemplo:

const soma = function(a, b) {
  return a + b;
};
console.log(soma(3, 7));

16. Arrow Functions (Funções de Seta)

Uma forma mais curta de declarar funções.

Exemplo:

const multiplicar = (a, b) => a * b;
console.log(multiplicar(4, 5));

17. Funções Concisas

Quando há apenas uma linha, o return pode ser omitido.

Exemplo:

const saudacao = nome => `Olá, ${nome}!`;
console.log(saudacao("Lucas"));
