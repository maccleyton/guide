O Document Object Model (DOM) é uma interface de programação para documentos HTML e XML. Ele representa a estrutura do documento como uma árvore de nós, onde cada nó corresponde a uma parte do documento. A manipulação do DOM permite interagir e modificar os elementos e conteúdos da página dinamicamente.

1. Seleção de Elementos

Para manipular elementos HTML, primeiro precisamos selecioná-los. JavaScript fornece vários métodos para isso.

1.1 document.getElementById

Seleciona um elemento pelo seu atributo id.

let titulo = document.getElementById("titulo");
console.log(titulo.textContent);

1.2 document.querySelector

Seleciona o primeiro elemento que corresponde ao seletor CSS especificado.

let paragrafo = document.querySelector("p");
console.log(paragrafo.innerHTML);

1.3 document.querySelectorAll

Seleciona todos os elementos que correspondem ao seletor CSS especificado, retornando uma NodeList. 

let itens = document.querySelectorAll("li");
itens.forEach(item => console.log(item.textContent));

1.4 Outros Métodos

  • document.getElementsByClassName: Seleciona elementos por classe.

    let elementos = document.getElementsByClassName("classe-exemplo");

  • document.getElementsByTagName: Seleciona elementos por nome da tag.

    let paragrafos = document.getElementsByTagName("p");

2. Manipulação de Conteúdo

2.1 innerHTML

Define ou obtém o conteúdo HTML de um elemento.

titulo.innerHTML = "<strong>Novo Título</strong>";

2.2 textContent

Define ou obtém o conteúdo textual (sem tags HTML).

titulo.textContent = "Novo Título";

2.3 innerText

Semelhante a textContent, mas considera o estilo CSS (por exemplo, elementos escondidos não serão retornados).

console.log(titulo.innerText);

3. Manipulação de Atributos

3.1 setAttribute

Define um novo atributo ou atualiza o valor de um atributo existente.

let link = document.querySelector("a");
link.setAttribute("href", "https://www.example.com");

3.2 getAttribute

Obtém o valor de um atributo.

console.log(link.getAttribute("href"));

3.3 removeAttribute

Remove um atributo de um elemento.

link.removeAttribute("target");

4. Manipulação de Classes

4.1 classList.add

Adiciona uma ou mais classes a um elemento.

titulo.classList.add("destaque", "texto-grande");

4.2 classList.remove

Remove uma ou mais classes de um elemento.

titulo.classList.remove("destaque");

4.3 classList.toggle

Adiciona a classe se ela não existir, ou remove se ela já estiver presente.

titulo.classList.toggle("oculto");

4.4 classList.contains

Verifica se o elemento possui uma classe.

console.log(titulo.classList.contains("destaque")); // true ou false

5. Manipulação de Estilos

5.1 style

Permite definir estilos inline diretamente em elementos.

titulo.style.color = "red";
titulo.style.fontSize = "24px";

5.2 Computed Styles

Obtém os estilos calculados de um elemento.

let estilos = getComputedStyle(titulo);
console.log(estilos.color);

6. Criação e Remoção de Elementos

6.1 createElement

Cria um novo elemento.

let novoParagrafo = document.createElement("p");
novoParagrafo.textContent = "Este é um novo parágrafo.";

6.2 appendChild

Adiciona um elemento como filho de outro.

document.body.appendChild(novoParagrafo);

6.3 removeChild

Remove um elemento filho.

let elementoParaRemover = document.getElementById("remover");
document.body.removeChild(elementoParaRemover);

6.4 replaceChild

Substitui um elemento filho por outro.

let novoTitulo = document.createElement("h2");
novoTitulo.textContent = "Novo Título";
document.body.replaceChild(novoTitulo, titulo);

7. Eventos

Eventos permitem que o JavaScript responda às interações do usuário.

7.1 addEventListener

Adiciona um ouvinte de evento a um elemento.

titulo.addEventListener("click", () => {
  alert("Você clicou no título!");
});

7.2 Remover Eventos

function alerta() {
  alert("Clicado!");
}
titulo.addEventListener("click", alerta);
titulo.removeEventListener("click", alerta);

7.3 Eventos Comuns

  • click: Disparado quando o elemento é clicado.

    titulo.addEventListener("click", () => {
  console.log("O título foi clicado.");
});

  • mouseover: Disparado quando o cursor do mouse passa sobre o elemento.

    titulo.addEventListener("mouseover", () => {
  console.log("Mouse passou sobre o título.");
});

  • mouseout: Disparado quando o cursor do mouse deixa o elemento.

    titulo.addEventListener("mouseout", () => {
  console.log("Mouse saiu do título.");
});

  • keydown: Disparado quando uma tecla é pressionada.

    document.addEventListener("keydown", (event) => {
  console.log(`Tecla pressionada: ${event.key}`);
});

  • keyup: Disparado quando uma tecla é liberada.

    document.addEventListener("keyup", (event) => {
  console.log(`Tecla liberada: ${event.key}`);
});

  • submit: Disparado quando um formulário é enviado.

    let form = document.querySelector("form");
form.addEventListener("submit", (event) => {
  event.preventDefault(); // Evita o envio do formulário
  console.log("Formulário enviado.");
});

  • change: Disparado quando o valor de um elemento é alterado.

    let input = document.querySelector("input");
input.addEventListener("change", () => {
  console.log("Valor alterado: " + input.value);
});
