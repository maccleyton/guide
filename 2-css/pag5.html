As cores hexadecimais são representadas por um código de 6 caracteres que começa com #, composto por números (0-9) e letras (A-F). Esse código define a intensidade de vermelho (R), verde (G) e azul (B) no formato #RRGGBB.

1. Estrutura da Cor Hexadecimal

Cada par de caracteres representa um valor entre 00 (mínimo) e FF (máximo):

  • #FF0000 → Vermelho
  • #00FF00 → Verde
  • #0000FF → Azul

A estrutura é a seguinte:

#RRGGBB
RR → Intensidade de vermelho (00 a FF)
GG → Intensidade de verde (00 a FF)
BB → Intensidade de azul (00 a FF)

2. Exemplos de Cores em Hexadecimal

Cor Código Hexadecimal
Preto #000000
Branco #FFFFFF
Vermelho #FF0000
Verde #00FF00
Azul #0000FF
Amarelo #FFFF00
Ciano #00FFFF
Magenta #FF00FF
Cinza #808080

3. Formas de Usar Cores no CSS

Você pode definir cores para fundos, textos e bordas com código hexadecimal:

body {
  background-color: #F0F0F0; /* Cor de fundo */
}
        
h1 {
  color: #FF5733; /* Cor do texto */
}
        
button {
  background-color: #008CBA; /* Cor do botão */
  border: 2px solid #005f73; /* Borda */
}

4. Formato Abreviado (#RGB)

Se os pares de valores forem iguais, você pode usar uma versão curta com 3 caracteres:

  • #FFF → Branco (#FFFFFF)
  • #000 → Preto (#000000)
  • #F00 → Vermelho (#FF0000)
  • #0F0 → Verde (#00FF00)
  • #00F → Azul (#0000FF)

Exemplo:

p {
  color: #333; /* Cor cinza escuro */
  background-color: #FAF; /* Rosa claro */
}

5. Cores com Transparência (#RRGGBBAA)

Você também pode adicionar um canal Alfa (AA) para transparência:

  • #FF000080 → Vermelho com 50% de opacidade
  • #000000C0 → Preto com 75% de opacidade

Exemplo:

div {
  background-color: #3498db80; /* Azul com transparência */
}

6. Conversão para Outros Formatos

Além de hexadecimal, você pode usar RGB e HSL para definir cores:

/* Hexadecimal */
color: #3498db;
        
/* RGB */
color: rgb(52, 152, 219);
        
/* RGBA (com transparência) */
color: rgba(52, 152, 219, 0.5);
        
/* HSL */
color: hsl(207, 70%, 50%);
