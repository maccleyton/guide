Guia Completo de CSS: Modos de Cor no CSS

  • Início
  • Voltar

No design e no desenvolvimento web, o uso correto das cores é essencial para criar interfaces harmoniosas e agradáveis. Abaixo, detalho diferentes paletas de cores, suas principais características e exemplos de cada uma.

O CSS permite especificar cores em diferentes formatos:

  • Hexadecimal (#RRGGBB ou #RGB) → Exemplo: #3498db
  • RGB (rgb(R, G, B)) → Exemplo: rgb(52, 152, 219)
  • RGBA (rgba(R, G, B, A)) → Exemplo: rgba(52, 152, 219, 0.5) (transparência)
  • HSL (hsl(H, S%, L%)) → Exemplo: hsl(207, 70%, 50%)
  • HSLA (hsla(H, S%, L%, A)) → Exemplo: hsla(207, 70%, 50%, 0.5)

Cada modo tem vantagens específicas, dependendo do contexto de uso.

6. Como Aplicar Paletas no CSS?

Você pode definir uma paleta de cores no CSS usando variáveis para facilitar a reutilização:

:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --background-color: #ecf0f1;
  --text-color: #2c3e50;
}
                
body {
  background-color: var(--background-color);
  color: var(--text-color);
}
                
button {
  background-color: var(--primary-color);
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
}
                
button:hover {
  background-color: var(--secondary-color);
}

6.3. Ferramentas Locais (Windows, Mac, Linux)

  • Windows: O Paint mostra cores em RGB.
  • Mac: O Digital Color Meter permite pegar cores da tela.
  • Linux: Ferramentas como Gpick ou KColorChooser podem ser usadas.

6.4. Lista de Cores Padrão do CSS

O CSS suporta 147 cores nomeadas que podem ser usadas sem precisar de códigos hexadecimais. Exemplo: 

color: red;      /* Vermelho */
color: blue;     /* Azul */
color: olive;    /* Verde oliva */
color: teal;     /* Azul esverdeado */
color: lavender; /* Lavanda */

Conclusão

Cada paleta de cores tem seu propósito e transmite uma sensação diferente. A escolha depende do objetivo do seu design:

  • Material Design → Interfaces modernas e limpas
  • Flat Design → Minimalismo e elegância
  • Pastel → Suavidade e delicadeza
  • Neon → Energia e impacto
  • Monocromático → Harmonia e simplicidade
