Guia Completo de CSS: Conceitos, Propriedades e Técnicas Avançadas

  • Início
  • Próxima

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas web, permitindo controlar cores, layouts e interatividade visual. Este guia aborda desde conceitos básicos até técnicas modernas para criar designs responsivos e acessíveis.

1. Fundamentos do CSS

1.1 Sintaxe Básica

A sintaxe do CSS é composta por seletores e declarações:

seletor {
  propriedade: valor;
}

Exemplo:

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
}

1.2 Tipos de Seletores

  • Seletores Simples:

    p {
  color: blue; /* Seleciona todas as tags <p> */
}
.classe {
  font-size: 16px; /* Seleciona elementos com a classe "classe" */
}
#id {
  margin: 10px; /* Seleciona o elemento com o id "id" */
}

  • Seletores Combinadores:

    div p {
  color: red; /* Seleciona <p> dentro de <div> */
}
a:hover {
  text-decoration: underline; /* Aplica estilo no estado hover */
}

1.3 Unidades de Medida

  • Relativas: %, em, rem, vw, vh.
  • Absolutas: px, cm, mm.

Exemplo:

h1 {
  font-size: 2em; /* Escala com base no elemento pai */
}

2. Box Model

2.1 Estrutura do Box Model

Cada elemento em CSS é representado como uma caixa com quatro áreas:

  1. Content: Área do conteúdo.
  2. Padding: Espaço interno.
  3. Border: Borda ao redor do padding.
  4. Margin: Espaço externo ao redor da borda.

Exemplo:

div {
  width: 200px;
  padding: 10px;
  border: 5px solid black;
  margin: 15px;
}

2.2 Propriedade box-sizing

Controla como o tamanho total da caixa é calculado.

  • content-box (padrão): Inclui apenas o conteúdo.
  • border-box: Inclui padding e borda no cálculo do tamanho.

Exemplo:

div {
  box-sizing: border-box;
}

3. Layouts Modernos

3.1 Flexbox

Flexbox facilita a criação de layouts responsivos e alinhamentos flexíveis.

Estrutura Básica:

.container {
  display: flex;
  justify-content: center; /* Alinha horizontalmente */
  align-items: center; /* Alinha verticalmente */
}

Propriedades Principais:

  • flex-direction: Define a direção dos itens (linha, coluna, etc.).

    .container {
  flex-direction: row;
}

  • gap: Controla o espaçamento entre os itens.

    .container {
  gap: 10px;
}

3.2 Grid Layout

Grid Layout é ideal para criar layouts bidimensionais mais complexos.

Estrutura Básica:

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

Propriedades Principais:

  • grid-template-areas: Define áreas nomeadas.

    .container {
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

  • grid-auto-rows: Define alturas automáticas para linhas.

    .container {
  grid-auto-rows: 100px;
}

4. Técnicas Modernas de CSS

4.1 Variáveis CSS

Permitem reutilizar valores em diferentes partes do código.

:root {
  --cor-primaria: #3498db;
  --espacamento: 16px;
}

button {
  background-color: var(--cor-primaria);
  padding: var(--espacamento);
}

4.2 Media Queries

Tornam o design responsivo para diferentes tamanhos de tela.

@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}

4.3 Animações CSS

Crie transições e animações suaves.

Transições

button {
  transition: background-color 0.3s ease;
}

button:hover {
  background-color: #2ecc71;
}

Animações com @keyframes

@keyframes girar {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}

.elemento {
  animation: girar 2s linear infinite;
}

5. Acessibilidade com CSS

5.1 Foco Visível

Certifique-se de que elementos interativos possuem um estado de foco claro.

a:focus {
  outline: 2px dashed #3498db;
}

5.2 Modo de Alto Contraste

Adapte estilos para usuários que precisam de maior contraste.

@media (prefers-contrast: high) {
  body {
    background-color: black;
    color: white;
  }
}
