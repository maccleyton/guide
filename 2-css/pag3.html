1. Seletores CSS

Os seletores do CSS são usados para identificar elementos HTML e aplicar estilos. Existem vários tipos de seletores:

1.1. Seletores Simples

São os mais básicos e incluem:

  • Seletor de Tipo: Seleciona elementos HTML pelo nome da tag.

    p {
  color: blue;
}

    (Todos os <p> terão cor azul.)

  • Seletor de Classe (.): Aplica estilos a todos os elementos com a mesma classe. 

    .destaque {
  font-weight: bold;
}

    (Qualquer elemento com class="destaque" terá o texto em negrito.)

  • Seletor de ID (#): Aplica estilos a um único elemento com um ID específico.

    #titulo {
  font-size: 24px;
}

    (Apenas o elemento com id="titulo" terá tamanho de 24px.)

  • Seletor Universal (*): Seleciona todos os elementos.

    * {
  margin: 0;
  padding: 0;
}

    (Remove margem e preenchimento de todos os elementos.)

1.2. Seletores Combinadores

Eles combinam diferentes seletores para criar regras mais específicas.

  • Seletor Descendente ( - espaço): Aplica estilos a elementos dentro de um determinado pai. 

    div p {
  color: red;
}

    (Todos os <p> dentro de <div> ficarão vermelhos.)

  • Seletor Filho (>): Seleciona elementos que são filhos diretos de outro elemento.

    div > p {
  color: green;
}

    (Apenas <p> que são filhos diretos da <div> ficarão verdes.)

  • Seletor Irmão Adjacente (+): Seleciona um elemento imediatamente após outro. 

    h1 + p {
  font-style: italic;
}

    (O primeiro <p> logo após um <h1> ficará em itálico.)

  • Seletor de Irmãos Gerais (~): Seleciona todos os elementos irmãos que vêm depois de um elemento específico.

    h1 ~ p {
  color: gray;
}

    (Todos os <p> depois de um <h1> ficarão cinza.)

1.3. Seletores de Atributo

Selecionam elementos com base nos atributos HTML.

  • Presença de Atributo:

    input[required] {
  border: 2px solid red;
}

    (Todos os <input> com required terão borda vermelha.)

  • Valor Específico (=):

    input[type="password"] {
  background-color: yellow;
}

    (Todos os <input> do tipo "password" terão fundo amarelo.)

  • Valor que Começa Com (^=):

    a[href^="https"] {
  color: green;
}

    (Links que começam com "https" terão cor verde.)

  • Valor que Termina Com ($=):

    img[src$=".png"] {
  border: 5px solid blue;
}

    (Imagens PNG terão borda azul.)

  • Valor que Contém (*=):

    div[class*="box"] {
  padding: 20px;
}

    (Qualquer <div> com "box" no nome da classe terá preenchimento de 20px.)

2. Atributos do CSS

Os atributos do CSS são propriedades que definem o estilo dos elementos. Aqui estão alguns dos principais grupos:

2.1. Atributos de Texto

  • color: Define a cor do texto.
  • font-size: Tamanho da fonte.
  • font-weight: Define a espessura do texto (normal, bold, etc.).
  • text-align: Alinhamento do texto (left, center, right, justify).

Exemplo:

p {
  color: navy;
  font-size: 18px;
  text-align: center;
}

2.2. Atributos de Fundo e Borda

  • background-color: Cor de fundo.
  • background-image: Define uma imagem de fundo.
  • border: Define uma borda.

Exemplo:

div {
  background-color: lightgray;
  border: 2px solid black;
}

2.3. Atributos de Layout

  • width: Largura do elemento.
  • height: Altura do elemento.
  • margin: Espaço externo ao redor do elemento.
  • padding: Espaço interno dentro do elemento.

Exemplo:

.box {
  width: 200px;
  height: 100px;
  margin: 10px;
  padding: 20px;
}

3. Elementos do CSS

Os elementos do CSS referem-se às tags HTML que podem ser estilizadas com CSS. Alguns dos mais comuns incluem:

  • Blocos de Texto: p, h1 - h6, span, strong, em
  • Blocos Estruturais: div, section, article, aside, header, footer
  • Formulários: input, textarea, button, select
  • Listas: ul, ol, li
  • Tabelas: table, tr, td, th

Exemplo de Aplicação no HTML e CSS:

<!DOCTYPE html>
<html lang="pt-br">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Exemplo CSS</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
        
  <h1 id="titulo">Título Principal</h1>
  
  <p class="destaque">Este é um parágrafo com destaque.</p>

  <div class="box">
    <p>Conteúdo dentro da div.</p>
  </div>

  <input type="text" required placeholder="Campo obrigatório">

</body>
</html>
/* Arquivo style.css */
body {
  font-family: Arial, sans-serif;
}
        
#titulo {
  color: darkblue;
  text-align: center;
}
        
.destaque {
  color: red;
  font-weight: bold;
}
        
.box {
  background-color: lightgray;
  padding: 20px;
  border: 2px solid black;
}

📝 Conclusão

Os seletores do CSS permitem estilizar elementos HTML com precisão. Os atributos controlam a aparência e o comportamento dos elementos, enquanto os elementos do CSS são as próprias tags HTML estilizadas.
