1. Introdução ao Dart

  • O que é Dart?

    • Dart é uma linguagem de programação desenvolvida pela Google, projetada para ser rápida, fácil de aprender e otimizada para o desenvolvimento de aplicativos móveis e web. Ele é usado no Flutter para criar interfaces de usuário de alto desempenho.

  • Instalação do Dart

    • Windows: O Dart é instalado como parte do Flutter SDK, portanto, ao configurar o Flutter, você também instala o Dart.
    • Acesse Dart - Install para obter instruções específicas, caso queira usar o Dart sem o Flutter.

  • Primeiro Programa em Dart

    • Escreva e execute seu primeiro programa simples em Dart:
    void main() {
print("Olá, Mundo!");
}

2. Configuração do Ambiente de Programação em Windows

Requisitos do Sistema

  • Sistema Operacional: Windows 7 ou superior (64-bit).
  • Espaço em Disco: Pelo menos 1,64 GB de espaço disponível para a instalação do Flutter SDK.
  • Ferramentas de Linha de Comando:
    • Git para Windows (necessário para clonar repositórios e gerenciar versões).
    • Um editor de texto (recomendado: VS Code ou Android Studio).

Passo a Passo para Instalação

1. Instalação do Git

  • Baixe o Git em "https://git-scm.com/" e instale-o. Durante a instalação, adicione o Git ao PATH do sistema.

2. Instalação do Flutter SDK

  • Baixe o Flutter SDK para Windows em "https://flutter.dev/docs/get-started/install".
  • Extraia o arquivo em uma pasta (exemplo: C:\src\flutter) e adicione o caminho C:\src\flutter\bin ao PATH do sistema.

3. Instalando o Android Studio

  • Baixe o Android Studio em "https://developer.android.com/studio".
  • Instale o Android SDK e o AVD (Android Virtual Device) para emulação de dispositivos Android.

4. Instalando o VS Code

  • Baixe o Visual Studio Code em "https://code.visualstudio.com/" e instale.
  • Instale os plugins "Flutter" e "Dart" no VS Code para facilitar o desenvolvimento.

5. Verificando a Instalação

  • Após a instalação do Flutter, execute:

    flutter doctor

    Isso verifica se tudo foi instalado corretamente e aponta problemas a serem corrigidos.

5. Introdução ao Flutter

  • O que é o Flutter?

    • Flutter é um framework de desenvolvimento de interfaces de usuário (UI) criado pela Google, que usa o Dart como linguagem principal. Ele permite o desenvolvimento de apps para Android, iOS, Web e Desktop com uma única base de código.

  • Instalação do Flutter

    • O Flutter SDK inclui o Dart, então ao instalar o Flutter, o Dart também será instalado.
    • Para instalar o Flutter, siga as instruções para Windows já descritas na seção de configuração do ambiente.

  • Primeiro Projeto em Flutter

    • Crie seu primeiro projeto com o comando:
    flutter create meu_primeiro_app
cd meu_primeiro_app
flutter run

6. Widgets Básicos do Flutter

  • Widgets em Flutter

    • O Flutter utiliza "widgets" para construir interfaces. Eles podem ser vistos como blocos de construção da interface do usuário.

  • Widget Stateless vs. Stateful

    • StatelessWidget: Não muda de estado após a construção.
    • StatefulWidget: Pode mudar de estado durante a execução. Exemplo de um contador:
    class Contador extends StatefulWidget {
@override
_ContadorState createState() => _ContadorState();
}
        		
class _ContadorState extends State<Contador> {
int _contador = 0;
@override
Widget build(BuildContext context) {
	return Scaffold(
	body: Center(
		child: Text("Contador: $_contador"),
	),
	floatingActionButton: FloatingActionButton(
		onPressed: () {
		setState(() {
			_contador++;
		});
		},
		child: Icon(Icons.add),
	),
	);
}
}
